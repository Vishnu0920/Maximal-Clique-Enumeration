Summary

This project implements and compares multiple maximal clique enumeration algorithms on large-scale real-world networks. Our key findings include:

  • Bron-Kerbosch Degeneracy showed the best overall performance.
  • Arboricity-based approach had the lowest memory consumption.
  • CLIQUE algorithm struggled with large-diameter networks.
  • ELS Algorithm showed efficient clique enumeration with moderate runtime.
