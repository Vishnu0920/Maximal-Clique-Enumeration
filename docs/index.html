Maximal Clique Enumeration

Implementation and comparison of three maximal clique enumeration algorithms.

Project Overview

This project implements and compares three algorithms for enumerating maximal cliques in graphs:

  • CLIQUE (Paper 1)
  • Bron-Kerbosch Degeneracy (Paper 2)
  • Arboricity-based (Paper 3)
