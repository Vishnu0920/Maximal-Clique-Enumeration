Datasets

Information about the graph datasets used for testing the maximal clique enumeration algorithms.

Dataset Selection Criteria

We selected a diverse set of graph datasets to evaluate the performance of the three maximal clique enumeration algorithms. The datasets vary in size, density, and structure to provide a comprehensive comparison across different graph types.

Size Variation

Graphs ranging from small (hundreds of vertices) to large (millions of vertices) to test scalability.

Density Variation

Both sparse and dense graphs to evaluate algorithm performance under different density conditions.

Structure Diversity

Real-world networks (social, biological, web) and synthetic graphs with controlled properties.

Clique Distribution

Graphs with varying distributions of maximal cliques to test algorithm efficiency.
